# Domain Checker Tools Pro 🔍

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/yourusername/domain-checker-tools)

Tools untuk mengecek Title, AMP, Canonical URL, dan Status HTTP dari sebuah domain secara lengkap dan akurat.

## ✨ Fitur

- ✅ **Single URL Check** - Cek satu URL dengan cepat
- ✅ **Batch URL Check** - Cek multiple URL sekaligus
- ✅ **Title Checker** - Ekstrak title tag dari halaman
- ✅ **Canonical URL** - Deteksi canonical URL
- ✅ **AMP Detection** - Deteksi AMP (AMP HTML, amphtml link, dll)
- ✅ **HTTP Status** - Status code dan response time
- ✅ **CORS Proxy** - Mengatasi masalah CORS dengan multiple proxy
- ✅ **Export Data** - Export ke CSV dan JSON
- ✅ **Responsive Design** - Tampilan yang responsif untuk mobile

## 🚀 Demo

Live demo: [https://domain-checker-tools.vercel.app](https://domain-checker-tools.vercel.app)

## 🛠️ Teknologi

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Backend**: Node.js, Vercel Serverless Functions
